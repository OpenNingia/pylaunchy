PyLaunchy
====================

PyLaunchy is an extension to [Launchy][Launchy]‘s plugin system, that allows developing Launchy plugins in the Python programming language.

Installation
-------
  - Windows: download and run the [Setup Program][InstallExe].
  - Other platforms: clone and build from source. Build instructions are specified in "Building.txt".
