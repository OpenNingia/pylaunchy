The following are needed for building PyLaunchy:
 * QT 4.4 (or any QT version compatible with Launchy)
 * Python 2.3+ (compatible with Boost.Python)
 * Boost 1.35
 
The following environemnt variables are needed:
 * QTDIR - path to QT root (e.g. C:\Qt\4.4.0)
 * PYTHON_ROOT - path to python root (e.g. C:\Python25)
 * BOOST_1_35_0 - path to Boost 1.35 top directory (e.g. D:\SDK\boost\boost_1_35_0)

The dev.bat file contains a script to start Visual Studio for PyLaunchy.
